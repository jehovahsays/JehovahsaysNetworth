# 📘 Operating the Subconscious

### **1. The Identity Handshake**
Always enter via `index.html`. The system checks for the `mev_human_verified` flag. If missing, you are redirected to the recovery terminal (`css.html`).

### **2. Using the Sovereign Search**
As you type, the search box converts "My Secret Note" into `My_Secret_Note`. This protects the internal storage keys and prevents URL-based injection.

### **3. The Recovery Protocol**
If the firewall triggers a lockout:
1. Navigate to the "Manual Reset" section in `css.html`.
2. Click **Verify & Restore**.
3. Type `RESTORE` when prompted to clear the breach flags.
