# Usage Guide — MEV

## 🧭 Website Navigation

### 1. The Perimeter (`index.html`)
The root page serves as the AI's "Visual Cortex" and firewall. If your IP is not in the `blackhole.dat` ledger, you will be granted access to the internal modules.

### 2. AI Perception Map (`/m/index.html`)
A Three.js celestial simulation. This represents the AI's perception of time and space.
- **Interaction**: Use the time slider to adjust the solar/lunar cycle.
- **Entry**: Click the "Play" link to transition into the player module.

### 3. Interactive Player (`/m/players.html`)
The core immersive experience.
- **Controls**: Use the on-screen joysticks (mobile) or WASD (desktop) to navigate the environment.
- **Exit**: Use the navigation links to return to the Map or the Root.
