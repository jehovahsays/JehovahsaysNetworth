🧠 MEV — Offline Voice AI

MEV is a browser-based offline assistant that learns from what you type and speaks responses aloud. It stores what you teach it locally in your browser using localStorage — no internet, no cloud, no tracking.

🚀 How to Use MEV

  1. Open MEV AI (or your index.html app).
  2. Teach it facts by typing a sentence such as “1+1 is 2”.
  3. Ask a question such as “What is 1+1?” — it will reply with “2”.
  4. It remembers everything you teach, even offline, until you clear your browser storage.
  5. Click the microphone button (optional future feature) to talk directly.

📴 Offline Capabilities

  • 🧠 Local memory — all facts are stored in localStorage.
  • 🔊 Voice output — uses your browser’s built-in speechSynthesis.
  • 💾 Service Worker caching — the app loads without internet.
  • 🧍 Works in private mode (with limited persistence).
  • 🔒 No external scripts, no remote servers, no data transmission.

🌐 Online Capabilities

  • When connected, the footer changes to “Online”.
  • Future versions can sync knowledge across devices using GitHub Pages or APIs.
  • Online mode may allow optional data export or leaderboard updates.

🧩 Browser Compatibility

  • ✅ Chrome, Edge, Brave — fully supported (speech + localStorage + offline cache).
  • ✅ Firefox — supported, though voice may differ slightly.
  • ✅ Safari / iOS — supported, but speech synthesis may pause on tab switch.
  • ⚠️ Private / Incognito — works temporarily, but data is erased when closed.

🧰 Security & Privacy

  • ✔ No network requests or API calls are made.
  • ✔ Uses only browser features: SpeechSynthesis, localStorage, Service Worker.
  • ✔ Passes key web security headers:
    • X-Frame-Options: DENY
    • Cross-Origin-Opener-Policy: same-origin
    • Permissions-Policy: geolocation=(), microphone=(), camera=()
  • 🔒 Data is stored only on your device. Clearing your cache or local storage will erase all learned data.

⚠️ Limitations

  • MEV cannot access the internet or real-time information when offline.
  • It only understands facts written in the form “X is Y”.
  • It does not yet handle complex logic, synonyms, or math parsing.
  • Speech recognition (voice input) may not work in all browsers offline.
  • Knowledge is local to your browser — not shared across devices.
