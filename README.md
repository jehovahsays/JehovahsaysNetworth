🧠 MEV: Multi-Environment Vanguard
MEV is an architectural model of an AI Brain, structured into Conscious (Public-facing Gateway) and Subconscious (Internal Security & Logic) layers. It is a folder-blind, offline-first personal wiki designed to function identically on localhost (iOS/PHPWin) and GitHub Pages.
📂 The MEV Ecosystem
The repository follows a hierarchical "Membrane" structure:
• Conscious Layer: The root index.html gatekeeper that handles human verification.
• Subconscious Layer: The /mev/ directory, protected by a Service Worker "Immune System."
• Perimeter Security: The /mev/blackhole/ tripwire designed to intercept and redirect unauthorized automation.
🚀 Key Features
• Zero-Latency Offline Mode: Powered by a v1.2.1 Service Worker.
• Circular Defense: A "Blackhole" redirect that ejects bots back to the primary root gatekeeper.
• MIME Awareness: Configurations (like phpwin.json) that allow mobile-hosted environments to treat code as educational text.
• Folder-Blind Pathing: Uses strictly relative pathing (./) to ensure mobility between local and public servers.
🛠️ Quick Start
1. Clone the repository to your local machine or iOS host.
2. Open the root index.html.
3. Interact with the splash screen to prove humanity and unlock the Subconscious logic.
