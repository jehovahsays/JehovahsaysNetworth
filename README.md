# MEV — Minimal Efficient Verifiable Wiki

MEV (Multi Edit Vandalism) is a single-page offline-first personal wiki designed to be **Minimal**, **Efficient**, and **Verifiable**. It works entirely within your browser, without needing a backend, and stores everything locally with optional encryption.

🧠 Perfect for journaling, planning, personal wikis, and secure offline note-taking.

---

## 🚀 Key Features

- ✅ **Single HTML File Architecture** — No external assets, all inline.
- 📴 **Offline-First** — Works without internet after first load.
- 🔐 **AES-GCM Encryption** — Optional PIN/passphrase-based local encryption.
- 🧩 **Modular Views** — Maps, Logs, Voice, Plays integrated into one app.
- 🧠 **AI Search Bar** — Filter pages via text or voice.
- 🧾 **Markdown-like Wiki Syntax**
- ⚙️ **CSP-Hardened** — Uses a strict Content-Security-Policy.
- 🛡 **No Tracking** — No cookies, telemetry, or third-party scripts.

---

## 🛠 How to Use

1. **Clone or download this repo:**
   ```bash
   git clone https://github.com/jehovahsays/mev
   cd mev
   ```

2. **Open `index.html` in your browser**. All features work offline.

3. **Start editing pages**. Search or create new pages in the sidebar.

4. **Set a PIN** for encryption (optional but recommended).
