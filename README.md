# 🌀 Localhost: The Sovereign Digital Subconscious

### **WHAT is this?**
This repository is a **Self-Hosted Cognitive Security Environment**. It is a serverless, local-first wiki and AI interface designed to live in the browser's "Subconscious" (Service Worker Cache). It ensures that your data, tools, and AI interactions remain under your absolute control.

### **THE CORE ARCHITECTURE (The Necker Cube)**
* **The Blackhole Sentinel (`index.html`):** A perimeter gatekeeper that identifies users. If a script or unauthorized bot is detected, it triggers a breach sequence.
* **The Conscious Interface (`wiki.html`):** The secure UI where humans interact with knowledge.
* **The Subconscious Memory (`sw.js`):** A Service Worker that caches the environment into the browser's persistent memory, enabling 100% offline sovereignty.
* **The Sovereign Logic (`a.js`):** The "Brain" that manages data without ever calling an external API or database.

### **WHY Localhost?**
Modern "Cloud" tech is **Digital Homelessness.** If you don't pay a subscription, your data is evicted. This project provides a permanent home in the Root.
