# localhost (MEV)

A sovereign, self-hosted offline brain. This is not a traditional website; it is a "Digital Subconscious" designed to live on `127.0.0.1` (::1), hardened against external interference and optimized for absolute privacy.

## 🌀 What is Localhost?
Localhost is a recursive, single-tier environment for thoughts, data, and AI interaction. It operates on the principle of **The Great Flattening**: all folders have been removed. The system is "folder-blind," meaning the entire brain lives in the root directory for maximum portability and resilience.

## 🧠 The User Experience
The experience is split into two distinct psychological states:
* **The External Conscious (index.html):** The entry gate. It is a minimalist terminal interface where you verify your humanity and intent before entering.
* **The Internal Subconscious (sw.js / a.js):** Once entered, the Service Worker caches the entire environment. Even if you lose internet, the brain stays alive in your browser's memory.

## 🛡️ Security by Design
* **The Humanity Handshake:** Access is restricted until a local "verification" status is established.
* **Recursive Redundancy:** The root contains `localhost.zip`. Inside that zip is another `localhost.zip`. This ensures your data exists in an infinite loop of backups.
* **Stealth Navigation:** The system automatically strips `.html` extensions from your address bar to maintain a clean, sovereign aesthetic.
